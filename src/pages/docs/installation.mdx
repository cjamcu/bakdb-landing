# Installing Bakdb

Bakdb is distributed as a single binary file, making it easy to install and run on various platforms.

## Download

Download the latest version of Bakdb for your operating system:

### x64
- [Linux (x64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_linux_amd64.tar.gz)
- [macOS (x64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_darwin_amd64.tar.gz)
- [Windows (x64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_windows_amd64.tar.gz)

### arm64
- [Linux (arm64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_linux_arm64.tar.gz)
- [macOS (arm64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_darwin_arm64.tar.gz)
- [Windows (arm64)](https://github.com/cjamcu/bakdb/releases/download/v0.0.1/bakdb_0.0.1_windows_arm64.tar.gz)

For more details, visit the [GitHub Releases](https://github.com/cjamcu/bakdb/releases) page.

## Quick Start

1. After downloading, extract the archive.
2. Run the binary:

```bash
./bakdb serve 
```

And that's it! A web server will be started with the following route:

- [http://localhost:8090](http://localhost:8090) - The Bakdb web interface.



    ## Deploy to production

    To deploy Bakdb to production, download the latest version of Bakdb for your operating system in the server and run the  executable with the `serve` command and your domain name (specifying a domain name will issue a Let's encrypt certificate for it))

```bash
./bakdb serve  yourdomain.com
```
