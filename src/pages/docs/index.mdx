# Welcome to Bakdb

Bakdb is a self-hosted, single-file solution that streamlines your database backup process across multiple database types.

## Key Features

- **Automated Backups**: Schedule regular backups to keep your data safe.
- **Multi-Database Support**: Compatible with MySQL, PostgreSQL, MariaDB.
- **Flexible Storage**: Store your backups locally or in the cloud.
- **User-Friendly Interface**: Manage your backups with an intuitive web interface.
