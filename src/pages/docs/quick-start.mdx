# Quick Start Guide

Get Bakdb up and running in just a few steps.

## Prerequisites

Before starting, make sure you have:
1. Downloaded the Bakdb binary for your system. If not, see our [Download Guide](./download).
2. Installed the necessary database clients. See our [Prerequisites](./prerequisites) page for details.

## Running Bakdb

1. Extract the downloaded archive.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the Bakdb binary.
4. Run the following command:

```bash
./bakdb serve 
```

And that's it! A web server will be started with the following route:

- [http://localhost:8090](http://localhost:8090) - The Bakdb web interface.
